# SQL-аналитика банковских транзакций (синтетические данные)

Проект: генерация синтетических данных о банковских транзакциях и их анализ с использованием SQL.

## Используемые технологии
- Python (pandas, numpy, Faker)
- SQLite
- SQL (оконные функции, CTE, сегментация RFM, когортный анализ)
- Git/GitHub

## Структура
- `generate_data.py` – генерация синтетических данных и загрузка в SQLite.
- `sql/` – SQL-запросы для анализа.
- `output/` – результаты запросов в CSV.

## Как запустить
1. Клонировать репозиторий.
2. Создать venv и установить зависимости (`pip install faker pandas numpy`).
3. Запустить `python generate_data.py`.
4. Выполнить запросы: `python scripts/run_queries.py` или вручную через `sqlite3`.

## Аналитические задачи
- Сегментация клиентов (RFM-анализ).
- Обнаружение подозрительных транзакций (ночные, большие суммы, зарубежные).
- Когортный анализ удержания.

Результаты визуализации (по желанию) можно добавить в output/plots.
